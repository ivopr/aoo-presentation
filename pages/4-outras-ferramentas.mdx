### Outras ferramentas

- ##### Grace: Ferramenta de plotagem 2D para o Sistema de Janelas X.[$^{[2]}$](https://plasma-gate.weizmann.ac.il/Grace/)
  - ###### Geração de gráficos para analises de simulações.
- ##### Celery: Fila de Tarefas Distribuida.[$^{[3]}$](https://docs.celeryq.dev/en/stable/)
  - ###### Execução de simulações de forma controlada e distribuida.
- ##### Flask: Framework Web para Python[$^{[4]}$](https://docs.celeryq.dev/en/stable/)
  - ###### Codificação do _backend_ em python para execução e geração de simulações.
- ##### NextJS: Framework React para a Web[$^{[5]}$](https://nextjs.org/)
  - ###### Codificação do _frontend_, possibilitando a disponibilização de uma interface intuitiva e amigável.
